Built using:

- Front-End : React
- Back-End : NodeJs, ExpressJs, MongoDB
  
In the /User-Hub/client, run:
### `npm install`
### `npm start`

In the /User-Hub/server, run:
### `nodemon server.js`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
